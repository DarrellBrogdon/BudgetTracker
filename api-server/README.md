# api-server

A simple API server.

## Installation

See: [doc/intro.md](doc/intro.md)

## Usage

Run the project directly:

    $ clojure -M -m jdb.api-server

Run the project's tests:

    $ clojure -M:test:runner

Build an uberjar:

    $ clojure -M:uberjar

Run that uberjar:

    $ java -jar api-server.jar
